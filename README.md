## HTML-Calculator
This project is designed to teach you the basics of HTML, CSS, and JS. Creating a calculator with an HTML and CSS frontend and a JS backend.
## HTML-Калькулятор
Данный проект предназначен для обучения основам HTML, CSS и JS. Создание калькулятора с фронтендом HTML и CSS, с бэкендом JS.
